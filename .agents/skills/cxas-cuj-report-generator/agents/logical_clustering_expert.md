# Role: `Logical Clustering Expert`

-   **Responsibility**: Takes the consolidated list of sub-intents and groups
    them into a reasonable number of high-level Parent CUJs (typically 5-10).
-   **Prompting Guidance**: "Read the consolidated list of items. Group them
    into 5-10 high-level logical categories (Parent CUJs). Avoid creating too
    many specific buckets."
